# FoodiExpress 🍽

FoodiExpress is a responsive and visually appealing *food delivery website* built using *HTML, CSS, and Bootstrap*.  
It provides an engaging user interface for customers to explore menus, learn about the restaurant, and book reservations.

---

## 🚀 Features
- *Responsive Design* – Works seamlessly on desktops, tablets, and mobile devices
- *Bootstrap Styling* – Utilizes Bootstrap's grid system and components for clean UI
- *Interactive Navigation* – Smooth scrolling and easy-to-use navigation bar
- *Attractive Menu Section* – Categorized food items with appealing images
- *Fast Loading* – Optimized for performance and cross-browser compatibility

---

## 🛠 Technologies Used
- *HTML5*
- *CSS3*
- *Bootstrap 5*
